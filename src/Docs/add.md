## Function Description

```
_c.add(augend, addend)
```

Adds two numbers.

## Function Technical Explanation

**Arguments**

augend (Number): The first number to add.

addend (Number): The other number to add.

**Returns**

(Number): Returns the sum of the augend and addend.
